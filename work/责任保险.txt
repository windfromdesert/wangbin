# 第一篇 基础篇

## 第一章：责任保险概述

### 第一节：责任风险

+ 法律责任的含义及分类
    - 法律责任，主要是法律责任中的民事责任，是责任保险产生的前提和基础
    - 法律责任的特点：
        1. 法律责任是违法行为、违约行为或法律规定的事由所引起的法律后果
        2. 法律责任要由法律明确规定
        3. 法律责任由国家强制力保证实施
    - 法律责任的分类
        1. 刑事责任
        2. 行政责任
        3. 民事责任
+ 责任风险
    - 责任风险的含义
        * 责任保险的可保风险是民事责任风险，因而责任风险指的就是民事责任风险。民事责任风险主要包括侵权责任和违约责任，侵权责任风险是责任保险的主要可保风险，违约责任风险只在个别情况下才成为责任保险的可保风险。
    - 责任风险的潜在财务后果
        1. 损害赔偿
        2. 抗辩费用
    - 责任风险的主要情形
        1. 机动车及其他交通工具
        2. 场所
        3. 企业经营
        4. 污染
        5. 动物
        7. 职业活动
    - 责任风险的特点
        1. 责任风险涉及的领域逐渐扩大，责任风险引起的赔偿金额愈来愈高
        2. 受害人的举证责任逐渐减弱，因而侵权人的民事责任风险逐渐加大

### 第二节：责任保险的概念

+ 责任保险的定义
    * 责任保险是以被保险人对第三者依法应负的赔偿责任作为保险标的的保险。
        1. 责任保险承保的是被保险人的民事赔偿责任风险
        2. 责任保险既可承保侵权责任风险，也可承保违约责任风险，但以承保侵权责任风险为主
        3. 被保险人对第三者责任的负担以法律法规的规定为依据
        4. 责任保险所承保的法定责任以作为受害人的第三者的存在为前提
        5. 责任保险所承保的民事责任必须是以损害赔偿形式承担的民事责任
+ 责任保险的保险责任和责任免除
    - 保险责任范围
        1. 被保险人造成他人人身伤害或财产损失依法应负的赔偿责任，保险人予以负责。
        2. 被保险人因其侵权行为（或承保违约责任时的违约行为）被提起诉讼或仲裁而支付的相关法律费用，保险人也负责赔偿。
    - 责任免除范围
        1. 不可抗力性的事件
        2. 行政行为或司法行为
        3. 核反应、核子辐射和放射性污染
        4. 污染
        5. 精神损害赔偿责任
        6. 被保险人的违约责任（合同责任）。经过特别约定的，可以承保，或者由专门的保险产品承保合同责任。
+ 责任保险与其他保险的区别 
    - 责任保险与人身意外伤害保险的区别
        1. 保险标的不同
        2. 被保险人对象不同
        3. 保险的目的不同
        4. 保险合同的性质不同
        5. 追偿权不同
    - 责任保险与财产保险的区别
        1. 物质损失只有在保险事故造成有形财产的损失时才产生保险赔偿，责任保险的保险赔偿既可以来源于有形财产损失，也可以来源于无形财产损失。
        2. 物质损失承保的仅仅是物质损失，责任保险经常包括对人身伤亡的赔偿责任。
        3. 物质损失保险的损失事件造成的是被保险人自己的财产损失，责任保险的损失事件造成的是被保险人以外的其他人的财产损失和人身伤亡。
    - 责任保险与保证保险的区别：保证保险与责任保险的区别在于，前者是代替投保人履行合同义务，后者是承担被保险人违反合同义务的后果。

### 第三节：责任保险的特征和作用

+ 责任保险的特征
    1. 责任保险具有双重补偿性
    2. 责任保险的保险价值不确定
    3. 保险费与责任限额呈非线性关系
    4. 保险责任的归属方式特殊
    5. 责任保险的理赔要求特殊
        - 赔偿责任确定的特殊性
        - 诉讼时效具有双重内涵
            1. 被保险人与受害第三者之间的侵权关系：诉讼时效一般为两年，在身体受到伤害、出售不合格商品事先未声明等情况下，诉讼时效期间为一年，自受害人知道或应当知道权利受到侵害之日起计算；
            2. 保险人与被保险人之间的保险合同关系：自被保险人知道或应当知道保险事故发生之日起计算，诉讼时效为两年。
+ 责任保险的作用
    1. 责任保险能够分散被保险人的意外风险，有利于维护受害人的合法权益
    2. 能降低社会成本，分担政府责任
    3. 有助现行法律制度的贯彻实施
    4. 能促进社会更加和谐

### 第四节：责任保险的分类

+ 侵权责任保险和违约责任保险
+ 公众责任险、雇主责任险、产品责任险、职业责任险和其他责任险
+ 团体客户责任保险和个人责任保险
+ 期内发生式责任保险和期内索赔式责任保险
+ 强制责任保险和自愿责任保险

### 第五节：责任保险的法律基础

+ 民事责任的概念
    1. 民事责任是民事主体承担的法律责任
        - 民事主体的基本特征就是其平等性，这是其区别于行政法律主体和刑事法律主体的根本。
        - 自然人、法人、其他组织
    2. 民事责任是违反合同义务或法定民事义务依法应承担的不利法律后果
        - 民事权利：人身权，财产权，知识产权。
        - 民事义务
    3. 民事责任不同于行政责任和刑事责任
    4. 民事责任的分类
        1. 侵权责任、违约责任和其他责任
        2. 单独责任与共同责任
            - 共同责任：按份责任、连带责任、补充责任
+ 侵权责任：是责任保险的主要承保风险
    - 侵权责任的概念
        1. 侵权行为是一种违法行为
        2. 侵权行为是一种有过错的行为
        3. 侵权行为以损害赔偿为主要责任方式
    - 规范侵权责任的法律：侵权普通法，侵权特别法。特别法优于普通法原则。新法优于旧法原则。特别法规定的一般侵权行为不得违反侵权行为一般条款原则。
    - 侵权责任的一般构成要件
        1. 违法行为：包括作为的违法行为和不作为的违法行为。
        2. 损害事实：对人身权利的损害事实[对人格权的损害、对身份权的损害]；对财产权利的损害事实；精神损害事实。
        3. 因果关系
        4. 过错：故意，过失。
    - 侵权责任的归责原则
        1. 过错责任原则
        2. 过错推定原则
        3. 无过错责任原则
        4. 公平责任的适用
    - 侵权责任的免责事由：侵权责任法规定：受害人过错、第三人过错、不可抗力、正当防卫、紧急避险；理论和司法实践中还有：职务授权行为、受害人同意、意外事件、自甘风险。
    - 侵权损害赔偿
        1. 赔偿权利人和赔偿义务人。赔偿义务人：直接加害人、替代责任人、按份责任人、连带责任人、补充责任人。
        2. 侵权损害赔偿：人身损害赔偿，侵害他人人身权益造成财产损失的赔偿，财产损害赔偿，精神损害赔偿。
+ 违约责任
    - 违约责任的概念
        1. 依据合同的性质和目的必须由合同的当事人所承担的义务
        2. 依据交易习惯所产生的默示义务
    - 违约责任的归责原则
        1. 无过错责任原则
        2. 过错责任原则
        3. 过错推定原则
    - 违约责任的免责事由：免责条款，不可抗力，合同法规定的其他免责事由[一类是受害人的过错，一类是自身属性]
    - 违约责任的承担方式：继续履行，采取补救措施，赔偿损失，支付违约金等
    - 违约损害赔偿
        1. 违约损害赔偿是因债务人不履行合同义务所产生的责任
        2. 违约损害赔偿以赔偿当事人实际遭受的全部损失为原则
        3. 违约损害赔偿有事先约定的赔偿数额计算办法的，按照合同的约定进行
        4. 违约损害赔偿原则上仅具有补偿性，而不具有惩罚性，目的在于填补债权人因违约而遭受的损害后果。
+ 民事责任的竞合
    - 民事责任竞合是指由于某种法律事实的出现而导致两种以上的民事责任产生，这些责任彼此之间是相互冲突的。
    - 处理民事责任竞合的原则
    - 侵权责任和违约责任的竞合
    - 侵权责任与刑事责任、行政责任的竞合
+ 诉讼时效
    - 普通诉讼时效：两年，自其知道或应当知道权利被侵害时起计算
    - 特殊诉讼时效：身体受到伤害；出售质量不合格商品未声明的；延付或者拒付租金的；寄存物品被丢失或者损毁的，自其知道或应当知道被损害情况之日起计算1年内。
    - 国际货物买卖合同和技术进出口合同争议提起诉讼或者申请仲裁的期限为4年，自当知道或应当知道其权利受到侵害之日起计算。
    - 诉讼时效期间的起算、中止和中断
    - 最长权利保险期限：从权利被侵害之日起超过20年的，人民法院不予保护。

## 第二章：责任保险合同

### 第一节：总则

+ 合同构成条款
+ 被保险人范围条款

### 第二节：保险责任

+ 责任损失条款
    1. 被保险人的侵权责任范围
    2. 被保险人的依法责任
    3. 保险责任归属方式：期内发生式；期内索赔式
    4. 保险事故
    5. 归责原则
+ 法律费用条款
    1. 法律费用的承保是责任保险区别于其他保险的一大特点
    2. 法律费用的限额通常有两种约定方式：一种是法律费用与被保险人对第三者的赔偿责任共用一个责任限额；另一种是法律费用单独设置一个责任限额。
    3. 第三者向被保险人起诉的内容还可能包括不属于保险责任的其他责任。
    4. 在国外，法律费用的额度非常高。
    5. 法律费用不是保险人必须承保的项目。

### 第三节：责任免除

+ 原因免责条款
+ 损失免责条款

### 第四节：责任限额与免赔额（率）

+ 责任限额：包括累计责任限额、每次事故责任限额、每次事故每人人身伤亡责任限额、每次事故每人医疗费用责任限额、法律费用责任限额。
+ 免赔额（率）
    - 一般针对医疗费和财产损失设置。

### 第五节：保险期间

一般是一年；保险期间与保险合同有效时间是两个不同的概念。一般来说，保险人签发保险单的时间就是保险合同生效的时间。

### 第六节：保险费

保险费率通常不会在保险条款中载明，而是在费率表中详细规定。

### 第七节：保险人义务

+ 签发保险单义务
+ 保险合同解除权行使期限
    - 保险人取得的合同解除权，自保险人知道有解除事由之日起，超过30日不行使而消灭。
    - 保险人在保险合同订立时已经知道投保人未如实告知的情况的，保险人不得解除合同；发生保险事故的，保险人应当承担赔偿责任。
+ 补充索赔证明和资料的一次性通知义务
+ 及时核定、赔付义务
    - 对是否属于保险责任作出核定要及时，30日是合理期限的法定最上限，如果个别案件情形复杂，双方当事人也可以约定合理期限
    - 对属于保险责任的，在与被保险人达成有关赔偿金额的协议后10日内，履行赔偿义务。
    - 对不属于保险责任的，应当自作出核定之日起3日内向被保险人发出拒赔通知书，并说明理由。
+ 先行赔付义务
    - 保险人自收到赔偿保险金的请求和有关证明、资料之日起60日内，对其赔偿保险金的数额不能确定的，应该根据已有证明和资料可以确定的数额先予支付；保险人最终确定赔偿数额后，应当支付相应的差额。

### 第八节：投保人、被保险人义务

+ 如实告知义务
+ 交付保费义务
+ 防灾义务
+ 危险程度显著增加通知义务
+ 损害事故通知义务
+ 损害赔偿请求通知义务以及协助抗辩义务
+ 单证提供义务
+ 协助追偿义务

### 第九节：赔偿处理

+ 确定被保险人对第三者赔偿责任的基础
    1. 三方协商
    2. 仲裁裁决
    3. 法院判决
    4. 保险人认可的其它方式
+ 向被保险人理赔的前提
    - 被保险人给第三者造成损害，被保险人未向该第三者赔偿的，保险人不负责向被保险人赔偿保险金。
+ 事故损失赔偿金额计算
+ 法律费用赔偿金额计算
+ 重复保险的处理
+ 被保险人索赔的诉讼时效
    - 被保险人向保险人请求赔偿的诉讼时效期间为2年，自其知道或者应当知道保险事故发生之日起计算。
    - 被保险人向保险人请求赔偿的诉讼时效
    - 受害第三者向被保险人索赔的诉讼时效

### 第十节：争议处理

首先由保险人与被保险人进行协商，协商不一致时，可以提交诉讼或仲裁。或依法向中华人民共和国人民法院起诉。

### 第11节：其他事项

其他事项主要是关于投保人中途退保和保险人中途解除保险合同的约定。

## 第三章：责任保险的历史与发展

### 第一节：责任保险的起源

依照通说，责任保险起源于法国。率先开办了雇主责任保险。

英国，在火车出现后，提供意外事故责任保险。

无过失责任的发展是与责任保险联系在一起的。责任保险制度成功地减轻并分散了加害人的负担，为无过失责任制度的发展提供了坚实的社会基础。

### 第二节：责任保险在国外的发展

+ 发达国家责任保险的发展历程
    - 第一阶段（19世纪末--20世纪30年代）：责任保险的萌芽阶段
    - 第二阶段（20世纪40年代--70年代）：责任保险获得广泛接受的阶段
    - 第三阶段（20世纪70年代至今）：挑战与机遇并存的现代发展阶段
+ 主要国家责任保险发展现状
    - 加拿大是全球第五大责任险市场
    - 英国是世界第二大责任险市场

### 第三节：责任保险在我国的发展

+ 主体结构
    - 截止2013年底，国内市场上有59家财产险公司，其中1家责任险专营公司（长安责任险公司），1家相互制保险公司（黑龙江阳光农业相互保险公司），36家中资公司，21家外资公司。
    - 2013年责任险市场前四大主体分别是人保财险、太保财险、平安财险、国寿财险。
    - 责任险市场集中度下降速度高于传统财产险及车险市场。
    - 外资公司责任险业务比重高于中资公司
+ 险种结构
    - 十一五前，主要集中在公众责任险、雇主责任险、产品责任险，老三样
    - 十一五后，各类险种均有所发展，赎金保险、艺术品保险、促销保险、电影保险、董监事及高级管理人员责任险、错误和遗漏责任险、环境污染责任险、自然灾害公众责任险等多种新产品也逐渐进入中国。

# 第二篇：产品篇

## 第一章：公众责任保险

### 第一节：公众责任风险与保险

+ 公众责任的定义：是指单位或个人（致害人）因自身侵权行为致使他人的人身或财产遭受损害，依法应由致害人承担的对受害人的经济赔偿责任。
+ 公众责任的法律依据：《民法通则》《侵权责任法》等
+ 公众责任风险：通常可以分为场所类公众责任风险和运营类公众责任风险。
+ 公众责任保险
    - 公众责任保险的概念
        * 广义的公众责任险意欲包含除雇主对雇员责任以及因拥有或使用汽车、飞机或船舶而产生的责任之外，个人和企业所面临的全部责任风险。
        * 狭义的公众责任险主要承保的是场所类公众责任风险
    - 公众责任保险的特点
        1. 受害方的不确定性
        2. 适用范围的广泛性
        3. 表现形式的多样性
    - 公众责任保险的作用
        1. 有利于维护人民群众的利益
        2. 有利于维护社会稳定
        3. 有利于辅助社会管理
    - 公众责任保险与第三者责任保险的区别
        1. 承保方式不同：公众责任险一般采用完全独立的保单承保；第三者责任保险则一般采用附加险的方式
        2. 理论体系不同：公众责任险列入责任保险理论体系，而第三者虽然本质上属于责任保险业务，但更强调与特定财产保险的相通性，在业务统计和考核上，也将其分别计入不同财产保险业务类别中。
    - 公众责任保险的发展
        * 始于1855年英国铁路旅客保险公司向铁路部门提供的承运人责任保险。

### 第二节：公众责任保险的主要产品

+ 场所类责任保险
    - 火灾公众责任保险；餐饮场所责任保险；校园方责任保险；
+ 道路客运承运人责任保险
    - 道路客运承运人责任保险的赔偿限额
        * 除另有约定外，一般设累计责任限额，每座责任限额，每人财产损害责任限额，每次法律费用责任限额和累计法律费用责任限额。一般每次法律费用责任限额按累计责任限额的5%确定，累计法律费用责任限额按累计责任限额的30%确定。
    - 道路客运承运人责任保险的附加险
        1. 附加司乘人员责任保险条款
        2. 附加三轮车责任保险
        3. 附加通勤车责任保险
        4. 附加精神损害赔偿责任保险
+ 供电责任保险
    - 常用附加险主要有三个
        1. 无过失（错）条款
        2. 索赔费用条款
        3. 自然灾害条款[不可抗力除外]
+ 环境污染责任保险
    - 保险责任
        1. 第三者人身伤亡及财产损失的责任
        2. 清污费用
        3. 法律费用
    - 责任限额
        * 每次事故第三者责任限额，每次事故每人人身伤亡责任限额和每次事故每人财产损失责任限额，每次事故每人医疗费用责任限额
        * 每次事故清污费用责任限额、累计责任限额
        * 每次事故法律费用责任限额、累计责任限额

## 第二章：雇主责任保险

### 第一节：雇主责任风险与保险

+ 雇主责任的含义：是指雇主对从事与雇佣有关活动的雇员依法应承担的民事赔偿责任。
    - 广义的雇主责任有两个层面的内容：一是雇员受害责任；二是雇员致害责任。狭义的雇主责任仅指雇员受害责任。
+ 雇主责任的法律依据：《民法通则》《劳动法》
+ 雇主责任的免责事由
    1. 受害人故意
    2. 不可抗力
+ 第三人致害的责任认定
    - 损害是因第三人造成的，第三人应当承担侵权责任。是第一位的赔偿义务人，雇主虽然承担连带责任或补充赔偿责任，只有在侵权人未赔偿、无力赔偿、赔偿不足或无法确定时才承担第二位的赔偿责任，雇主赔偿以后可以向直接侵权人追偿。
+ 雇主责任风险与保险
    - 我国的雇主责任保险模式
        * 雇主责任的大小以及雇主责任的赔偿依据，应该按《工伤保险条例》标准计算。
+ 雇主责任保险与其他责任类似保险产品的区别
    - 雇主责任险与工伤保险的区别
        1. 保险性质不同。工伤保险是我国社会保险的一种，具有社会强制性，雇主责任险属于商业保险，由企业自愿投保。
        2. 保险责任不同。雇主责任险的保险责任一般采取非列明方式；工伤保险的保险责任采取细分式。
        3. 投保方式不同。雇主责任险没有限定必须采用记名投保；工伤保险一般必须采用记名的投保方式。
        4. 赔偿方式不同。雇主责任险的赔偿先赔付给企业，由企业赔付给雇员；工伤保险规定由工伤保险基金支付的部分，直接支付给雇员。
        5. 计费基础不同。工伤保险通常是按照企业工资总额计收保费，同时，雇主、雇员都要承担一定比例的保险费；雇主责任保险既可按照工资总额，也可按照责任限额计收保费，完全由雇主承担保险费。
        6. 赔偿限额标准不同。雇主责任险一般是在赔偿限额以内据实赔付，购买的限额越高，则同等伤残等级下可以获得的补偿则越高；工伤保险则是按照平均工资水平在法律规定的限额内进行赔偿。
        7. 自担风险程度不同。雇主责任险中，雇主可以按照投保意愿选择无免赔额的承保条件，一旦出险，在责任限额内雇主可以获得更多的保险保障。工伤保险中，企业自担风险是法律已经约定的，应由用人单位支付的部分有：治疗工伤期间的工资福利；五级、六级伤残职工按月禽的伤残津贴；终止或者解除劳动合同时，应当享受的一次性伤残就业补助金。如果同时办理了工伤保险，又投保了雇主责任险，就存在赔偿责任分摊的问题。保险公司在雇主责任险保单条新款下仅承担差额责任。新版雇主责任险，保险赔偿责任修订为定额赔偿，只要符合保险责任范围，即按照保险条款约定的责任限额进行赔偿。
    - 雇主责任险与意外险的区别
        1. 需求的产生基础不同。人身意外伤害保险产生的基础是人们对自己生命、身体生产保障需求；雇主责任保险产生的基础是法律制度。
        2. 保险标的不同。人身意外保险以人的生命或身体为保险标的。而责任保险的保险标的是被保险人在法律上对他人的损害赔偿责任，是一种无形的法律风险。
        3. 投保目的不同。人身意外伤害保险是为了补偿因意外事故造成被保险人自己的生命、身体伤害的直接损失；责任保险的受益者有两个，被保险人和受害人，实际受益人是第三者（受害人）。
        4. 保障项目不同。责任险可以承担企业依法应承担的搜救费用（安全生产责任险）、法律费用，以及员工住院期间的床位费、陪护费、伙食费、取暖费、空调费等。意外险不承担上述责任。雇主责任险对于职业病、抢险救灾和公益活动的意外伤害、工作期间突发疾病48小时死亡、服役时旧伤复发等情形也承担赔偿责任。意外险一般对职业病、非意外伤害事故不承担责任。

### 第二节：主要产品

+ 主要产品：雇主责任保险1995年版、1999年版、2004年版、工伤责任保险、建筑施工企业雇主责任险、安全生产责任险、教职员工校园方责任险、渔工责任险、职业院校实习生责任险等。
+ 保险责任
    - 主要条款之间的差异和特点
        1. 1995年版雇主责任险。一是保险责任为非列明责任，二是医疗费用在伤亡限额范围内赔偿，与伤亡赔偿共用限额，三是没有限制医疗费用赔偿范围，四是死亡、残疾赔偿限额按若干月工资确定。
        2. 1999年版雇主责任险（建筑施工雇主责任险）。一是保险责任为非列明责任。二是医疗费用为单独投保的单项限额，没有投保则不予给付。三是医疗费用有限制。四是对死亡、残疾赔偿采用定额赔付方式。
        3. 2004年版雇主责任险。一是保险责任为列明责任。二是医疗费用为单独投保的单项限额，没有投保则不予赔付。三是医疗费用几乎没有限制。四是对死亡、残疾赔偿采用限额内据实赔付方式，应该按照工伤保险条例计算雇主应承担的赔偿责任。
        4. 工伤责任保险。保险责任为《工伤保险条例》中规定的工伤事故，负责赔偿《工伤保险条例》中规定应由用人单位负责赔偿的部分。
+ 计费基础及赔偿限额
    - 2004年版雇主责任保险：限额不低于3万；每人医疗费用责任限额不超过每人伤亡责任限额的50%并且不高于5万元人民币，法律费用责任限额为伤亡责任限额的20%。
    - 建筑施工企业雇主责任险：以建筑项目投资总额为计算保费的基础。每次事故每人人身伤亡责任限额分九档，范围为3万元--40万元，累计责任限额为工程造价的15%，每次事故每人医疗费（包括抢救费）责任限额以每次事故每人人身伤亡责任限额的40%来确定。
+ 安全生产责任险简介

## 第三章：产品责任保险

### 第一节：概述

+ 产品与产品责任
    - 产品责任：广义的产品责任既包括因产品缺陷造成的损害应承担的民事赔偿责任（侵权责任），又包括因产品质量不合格所导致的合同责任（违约责任）。狭义的产品责任只包括侵权责任。
    - 产品缺陷
    - 产品责任构成的要件：一是须有缺陷产品；二是须有人身、财产损害事实；三是产品缺陷与损害事实之间须有因果关系。
    - 产品侵权责任与产品质量违约责任的区别
+ 产品责任法概述
    - 产品责任归责原则的发展过程：合同关系责任；疏忽责任；担保责任；严格责任四种归责原则。
    - 我国目前尚无专门的产品责任法，《民法通则》《侵权责任法》《产品质量法》《消费者权益保护法》等构筑起我国产品责任法律制度的基本框架。
    - 产品责任承担的主体：我国，既包括生产者也包括销售者，但没有对其范围作出规定。

### 第二节：产品责任保险

+ 产品责任保险的发展
+ 我国产品责任保险的主要产品
    - 产品责任保险条款及商业综合责任保险条款
    - 人保条款规定损害赔款与法律费用使用一个限额，避免高额法律带来的额外赔偿。
    - 附加险条款
        1. 以索赔提出为基础条款
        2. 发现期条款
        3. 增加被保险人条款
        4. 产品完工条款
        5. 限制性条款
+ 与产品责任相关联的保险产品介绍
    - 产品召回保险
    - 产品质量保证保险

## 第四章：职业责任保险

### 第一节：职业责任风险与保险

+ 职业责任与职业责任风险
    - 职业责任含义
        1. 具有专业服务的资格
        2. 具有高度的专门性
        3. 社会对专业人士存在特殊依赖性
    - 职业责任保险的法律依据：违约责任；侵权责任；责任竞合。
    - 职业责任风险：因工作上的失误导致的民事损害赔偿责任风险
+ 职业责任保险：是指承保各种专业技术人员因在从事职业技术工作时的疏忽或过失造成委托方经济利益损失的保险。按照惯例，基本上都以“索赔提出”作为承保基础。
+ 职业责任保险的历史发展

### 第二节：职业责任保险主要产品 

+ 医疗责任保险
    - 责任限额：医疗责任每人责任限额；精神损害每人责任限额；医疗责任累计责任限额；法律费用每次事故责任限额、法律费用累计责任限额。除另有约定外，精神损害每人责任限额为医疗责任每人责任限额的30%，并包含在医疗责任每人责任限额之内。
+ 建设工程设计责任保险
    - 责任限额：一般包括累计赔偿限额、每次事故赔偿限额和每次事故第三者人身伤亡每人赔偿限额。
    - 建设工程设计责任保险通常按照年度承保，承保基础是“索赔提出式”。
    - 单项工程设计责任保险则仅承保在保险单中载明的单个建设工程项目，通常保险期间大于1年。
+ 律师职业责任保险
    - 赔偿限额：每次事故赔偿限额和累计赔偿限额，累计赔偿限额一般为每次事故赔偿限额的2倍。法律费用每次事故赔偿金额包含在每次事故赔偿限额之内。
+ 注册会计师责任保险
    - 赔偿限额：每次事故赔偿限额，累计赔偿限额，也可以不设每次事故赔偿限额。
    - 附加险：附加账册、文件丢失险条款；首次投保追溯期扩展条款。

## 第五章：其他责任保险

### 第一节：物流责任保险

+ 物流责任保险概述
    - 物流的基本功能：运输功能，储存功能，装卸搬运功能，包装功能，流通加工功能，配送功能。
    - 物流风险：与托运人之间可能产生的风险；与分包商之间可能产生的风险；与信息系统提供商的合同责任风险；第三者责任风险。
+ 主要保险产品
    - 物流责任保险：是物流企业在经营物流业务过程中发生物流货物损失时，依法应由被保险人承担的经济赔偿责任以及相关费用损失。
    - 国际货运代理人（提单）责任保险。国际货运代理是指代表进出口商完成货物的装卸、储存、安排内地运输、收取货款等业务的代理机构。
+ 无船承运业务经营者保证金责任保险
    - 无船承运经营业者，即以承运人身份接受货主（托运人）的货载，同时以托运人身份委托班轮公司完成国际海上货物运输，根据自己为货主设计的方案路线开展全程运输服务的经营业者。

### 第二节：董事责任保险

+ 董事责任保险概述
    - 美国是董事责任保险制度的发源地。
+ 董事责任保险产品的主要内容
    - 保险责任：通常采用“一切险”加列明除外的方式，保险责任通常包括两个主要部分：1.董事个人责任保险[SIDE A]，其保障的是董事个人的财产，一般不设免赔额；2.公司补偿保险[SIDE B]，其保障的是公司的财产，一般要设免赔额。
    - 被保险人一般是指担任被保险公司的董事或高级管理人员或类似职务的自然人。
    - 一般为了控制风险，抗辩费用与损害赔偿金通常共用同一限额。
    - 以“索赔提出”为承保基础。
    - 是否属于保险责任三个要素：
        1. 不当行为何时发生
        2. 第三者何时向被保险人/被保险公司提出赔偿请求
        3. 被保险人何时将赔偿请求通知保险人。保单通常自动提供30-60天的发现期。
    - 附加险条款
        1. 继承人、遗产管理人及法定代理人条款
        2. 共同财产条款
        3. 外部董事条款
        4. 非执行董事条款
        5. 退休董事及高级管理人员条款
        6. 雇佣赔偿请求条款
        7. 发现期条款
        8. 附加公司证券责任保险条款

### 第三节：建筑工程质量保险

+ 建筑工程质量保险概述
    - 在国际上，建筑工程质量保险又称为“潜在缺陷保险”。简称IDI，或“十年期责任保险”。
    - 所谓潜在缺陷是指由于设计错误、施工工艺或建筑材料引起的缺陷和建筑物竣工验收（建筑质量检查机构颁发完工证书）时未发现的缺陷。
    - 1978年，法国制订了《斯比那塔法》。
    - 法国建设工程质量保险体系：建筑工程质量保险；职业责任险（勘察、设计、质量检查监督机构）；和承包商责任险。
    - 法国等国开展建设工程质量保险的经验借鉴
        1. 建立各方利益相互制约的关系
        2. 政府的管理方法科学、合理
        3. 工程质量协会
        4. 法国、西班牙和意大利在制定建筑物的使用功能和舒适度的标准时，均遇到了难以界定的问题。
    - 我国建筑工程风险管理现状
        * 建筑行业相关制度：建筑法；建设工程质量管理条例；配套制度。
+ 建筑工程质量保险产品介绍
    - 投保人。是建筑开发商，主要适用的建筑类型为住宅商品房及写字楼工程。道路、隧道等建设工程由于风险较为特殊，并不适用于本保险条款。
    - 保险责任：承保的主要风险就是地基基础工程及主体结构工程的质量事故。
    - 保险期间：建筑工程质量保险的保险期间的起始日期为建筑物竣工验收通过1年之后，至建筑物竣工验收通过后11年，保险人承担保险责任的期间共10年时间。
    - 责任限额：累计责任限额、单位建筑面积责任限额及每张保险凭证责任限额。责任限额确定时应以工程造价乘以一定的通货膨胀系数来计算。

## 第六章：个人责任保险

### 第一节：个人责任风险与保险

+ 个人责任的定义
    1. 个人直接侵权责任
    2. 个人违反法律规定的义务
    3. 个人或家庭所有静物责任
    4. 个人或家庭饲养的动物责任
+ 个人责任保险法律依据：《民法通则》《侵权责任法》
+ 个人责任风险
+ 个人责任保险：是以在保险期间内因个人或家庭成员的过失而发生的意外事故并造成第三者人身损害或财产损失，依法应负的民事赔偿责任为保险标的的保险。
    - 个人责任习惯上列为公众责任保险范畴
    - 个人责任保险存在公众责任保险、雇主责任保险和职业责任保险的交叉

### 第二节：个人责任保险主要险种

+ 个人综合责任保险：是以被保险人因过失造成第三者的人身伤亡或财产损失，依法应由被保险人承担的赔偿责任为保险标的的保险。
+ 居家责任保险：是以被保险人或其家庭成员在保险单载明的被保险人住址内，因过失造成第三者人身伤亡或财产损失，依法应由被保险人承担的赔偿责任为保险标的的保险。
+ 家庭雇佣责任保险：是以被保险人雇佣的家政服务人员因从事被保险人的家务工作而遭受意外，所致人身伤亡，依法应由被保险人承担的经济赔偿责任为保险标的的保险。
    - 附加险：附加家政服务人员第三者责任保险；附加家政服务人员忠诚保险。
+ 动物饲养责任保险：是以被保险人饲养或管理的动物造成第三者人身伤亡或财产损失，依法应由被保险人承担的赔偿责任为保险标的的保险。
+ 监护人责任保险：是以被保险人的监护对象（被监护人）造成第三者人身伤亡或财产损失，依法应由被保险人承担赔偿责任为保险标的的保险。
+ 电动自行车第三者责任保险：是以被保险人在使用保险电动自行车过程中发生意外事故，致使第三者遭受人身伤亡或财产直接损毁，依法应由被保险人承担的经济赔偿责任为保险标的的保险。

# 第三篇：定价篇

## 第一章：定价原理

### 第一节：费率组成

+ 风险保费：又称纯保费，是每个风险单位的平均赔付成本。
    - 风险单位的确定：合理性；易于确定；反映变化；历史经验。
    - 最终损失的确定：一般建立在历史统计数据的基础上。
    - 出险频率与案均赔款要分开来考虑。主要原因在于，影响出险频率与案均赔款的因素各不相同。此外，在保单中引入免赔额和超赔再保险时，必须要弄清楚案均赔款的分布情况。
    - 新开发险种测算风险保费的方法
        1. 参考类似产品的数据
        2. 参考其他公司的数据
        3. 参考行业数据
        4. 参考国外数据
        5. 依靠经验
        * 对以上方法定出的产品风险保费仍然需要定期跟踪测算，对结果进行修正。
+ 费用附加
    - 费用构成：理赔费用；承保费用[佣金和手续费、业务管理费用（包括员工工资、办公室租金、水电费、保险单证印制费等）、其他获得费用（如广告宣传费用）、营业税，保险保障基金与年费]；
    - 费用分配：直接营业费用与间接营业费用。间接营业费用存在着不确定性。间接营业费用分摊的几种分配标准：
        1. 按员工工资金额分配
        2. 按员工数量分配
        3. 按直接营业费用的比例分配
        4. 按场所使用面积分配
        5. 按保费收入的比率分配
        6. 专案分析
        7. 按工时分配
+ 风险及利润附加
+ 保险价格各部分的关系
    - 保险价格包含三个部分：纯保费，费用附加，风险及利润附加
    - 盈亏平衡价格：纯保费，费用附加（手续费）
    - 成本价格：纯保费，费用附加（不含手续费）

### 第二节：费率厘定基本方法

+ 基准费率厘定
    - 风险保费法
    - 损失率法
    - 两种方法的数据环境要求不同，因此在不同的条件下，应采用不同的定价方法：
        1. 风险保费法建立在每个风险单位的损失基础之上，它需要严格地定义风险单位，如果风险单位不易认定，或风险单位之间有差异，则风险保险法不适用。
        2. 损失率法是通过对当前费率进行调整实现定价的，它需要当前费率和保费历史记录。因此，损失率法不能用于新业务的定价。
        3. 当前费率难以计算时，风险保费法更为适用。
+ 级别费率厘定
    - 确定级别费率的关键是确定各级别费率与基础费率之间的相对关系，用相对数来表示，即指示相对数。
+ 广义线性模型：是一种基于风险分类的定价方法。也是目前公认的非寿险产品定价通用的指导性方法。GLM通过构造包含各种风险要素的定价模型来体现损失因素和损失赔付的各种特性。
+ 个体风险定价：有两种基本类型：未来法和过去法。
    - 未来法的基本原理是依据过去的损失经验确定将来的风险成本。采用未来法的个体风险定价系统有三种类型：表定费率法；经验费率法；复合费率法。
    - 过去法的基本原理是根据当期的实际经验确定当期的最终成本。其最大的特点是事后评价，即在保险期限结束后，根据过去的实际情况确定并调整保险费。过去法多用于劳工赔偿险和雇主责任险的费率厘定。过去法要求投保人在保险期限开始时，先缴纳预付保费。
    - 与未来法相比，过去法与个体的当前风险特征更为一致，但同时采用过去法确定的风险成本稳定性较差，不同的年景保费差别会比较大，而且过去法周期较长，一般要在很长时间后才能最终确定风险成本。

### 第三节：影响定价的主要因素

+ 未决赔款
+ 历年赔付的变化趋势
+ 保单条件的变化
+ 核保政策的变化
+ 理赔管理的变化
+ 数据质量
+ 公司经营政策

## 第二章：责任保险定价

### 第一节：责任保险定价特点

+ 责任险限额增长因子的调整
    - 责任险的责任限额总体看可以分为两类：单一限额和复合限额。责任限额因子是指不同档次限额之间的相对关系。责任限额与限额因子不是线性关系。
    - 测算方法：经验数据法和损失分布法
+ 期内索赔制与事故发生制：从定价角度考虑，如果赔款有逐年增长的趋势，则事故发生制保单的成本比期内索赔制保单成本要高。在基于相同的赔款增长趋势进行定价时，如果赔款的增长趋势突然发生改变，则其内索赔制保单的定价结果更接近真实值。

### 第二节：责任保险主要险种定价

+ 公众责任保险。考虑的因素
    1. 保险人的业务性质产生损害赔偿的可能性大小
    2. 被保险人的风险类型
    3. 被保险人的管理水平与管理效果
    4. 被保险人以往事故和赔偿记录
    5. 承保区域范围的大小
    6. 司法管辖范围等
+ 雇主责任保险：年度费率，一般根据被保险人的工资总额、工作地址、行业性质、风险程度以及被保险人选定的赔偿限额来确定。
+ 产品责任保险。考虑的因素
    1. 产品的特点和可能对人体或财产造成损害的风险大小
    2. 产品的数量
    3. 承保的地区范围，即承保产品销售区域的大小。
+ 职业责任保险。应着重考虑的因素：
    1. 职业类别
    2. 工作场所
    3. 业务数量
    4. 被保险人单位的性质，如营利性或非营利性
    5. 被保险人及期雇员的专业技术水平
    6. 被保险人及其雇员的工作责任心和个人品质
    7. 被保险人职业责任事故的历史统计资料及索赔、理赔情况
    8. 赔偿限额、免赔额及其他承保条件
+ 核保经验系数：定价时，除了确定基准费率之外，还可以根据核保人的经验对基准费率进行调整。对基准费率进行调整的系数统称为核保经验系数。

# 第四篇：承保篇

## 第一章：公众责任保险

### 第一节：风险调查与评估

+ 风险调查
    1. 被保险人营业场所
    2. 投保区域范围
    3. 被保险人营业性质
+ 风险评估
    1. 经营业务种类
    2. 被保险人或其聘用的管理者的管理水平
    3. 投保场所及周围环境
    4. 保险地点的安全设施及应急手段如何
    5. 与周围消防、医院距离远近程度如何
    6. 与投保行业有关的民事损害赔偿责任的法律规定有哪些
    7. 被保险人以往事故记录。
    8. 责任限额的高低和免赔额
    9. 被保险人有无健全的安全生产制度并予以落实，防灾防损情况如何
    10. 被保险人是否有其他特殊保障要求。

### 第二节：风险控制

+ 控制逆选择：要仔细询问投保人历史是否发生赔付或相关风险事故，要求投保人盖章确认已如实告知，同时，应通过多种第三方渠道了解该投保人相关风险信息。另外，严格按照公司颁布的行业费率表进行审核。
+ 控制保险责任
    1. 限额控制
    2. 免赔条件控制
    3. 责任免除控制
    4. 扩展责任/附加条款控制
+ 控制人为风险：道德风险和心理风险

### 第三节：承保方案

+ 承保方案应包括的内容
    1. 投保人名称、地址和电话等
    2. 被保险人
    3. 营业性质
    4. 营业处所地址
    5. 投保区域范围
    6. 责任限额及免赔额
    7. 附加险责任限额
    8. 费率
    9. 保险费
    10. 保险期间
    11. 备注
    12. 司法管辖权
+ 其他公众责任险的承保方案制定
    - 道路客运承运人责任险
        1. 被保险人资质及基本情况
        2. 营运地域范围、运输路线
        3. 投保车辆情况
        4. 承运司机资质及相关风控措施
        5. 是否已投保相关险种
        6. 责任限额及免赔额
        7. 保险费
    - 校园方责任险
        1. 被保险人性质
        2. 被保险人注册学生人数
        3. 被保险人学校风险管理情况
        4. 责任限额及免赔额
        5. 保险费
    - 供电责任险
        1. 被保险人性质
        2. 被保险人风险管理情况
        3. 责任限额及免赔额
        4. 保险费
    - 物业管理责任险
        1. 物业管理企业的服务区域面积、服务业主的户数
        2. 服务区域内下水管道已使用年限及是否定期检修
        3. 服务区域内有无健身器材及其是否定期检修
        4. 物业管理场所中包含的电梯或升降机、游泳池、锅炉、广告牌等风险因素

### 第四节：案例介绍

## 第二章：雇主责任保险

### 第一节：风险调查与评估

+ 雇主责任保险风险调查
    1. 行业类别
    2. 作业场所的设计是否符合卫生标准和要求
    3. 工种类别
    4. 员工工作场所的安全设施及应急抢救措施或手段如何？
    5. 被保险人的管理人员结构及水平如何
    6. 劳资关系如何
    7. 被保险人以往的事故记录。
    8. 被保险人以往的保险记录，是否有被保险公司解除保险合同的情况发生；被保险人是否参加社会医疗保险、工伤保险
    9. 劳动合同（或雇佣合同）中被保险人对所聘用员工因为意外或患职业病而造成伤残、死亡等规定的赔偿原则及限额高低。
    10. 承保区域范围有无特殊要求
    11. 被保险人有无扩展责任要求及与扩展责任相应的风险如何
+ 雇主责任保险风险评估
    1. 微观方面的评估
    2. 宏观方面的评估

### 第二节：风险控制

+ 一揽子业务形式
+ 建立与投保人的利益共同分担机制：设定一定比例的免赔率或固定的免赔额。
+ 对投保客户的条件限制
    1. 选择合法性和规模性的客户
    2. 保险限额的控制：一般而言，行业类别、员工职业类别风险高或员工工资低的企业，保险限额应低些；相反，则保险限额可设置高些。
    3. 设置合理的保险费率。
    4. 对劳动合同的约定。限制企业因为保险而随意扩大责任的行为。
+ 承保方式的选择
    - 两种雇主责任保险方式，一种是近似工伤保险的雇主责任保险，另一种是作为工伤保险补充方式的雇主责任保险，又称工伤责任险或雇主工伤补偿责任保险。
    - 近似工伤保险的雇主责任险依计收保费的不同，主要可分为三类：第一类是以企业雇员的年工资为计收保费依据；第二类是以责任限额为计收保费基础；第三类是按工程项目造价计收保费；此外，还有按行业类别不同按限额计收保费的雇主责任保险。
+ 足额投保及分保：要所有员工投保，避免选择投保，尽量做到记名投保；同时，为控制风险，要做好分保工作，以有效分散风险。对超过自留额的业务，要做好临分。

### 第三节：承保方案

+ 投保人与被保险人
+ 承保区域
+ 保险保障范围
+ 赔偿限额和免赔额
    1. 按照若干个月工资标准设定赔偿限额
    2. 根据需求直接设定每人赔偿限额
    3. 按照工程造价标准设定赔偿限额
+ 保险期限
+ 保险费率
    1. 投保企业的行业类别及员工的工程
    2. 被保险人的过往情况
    3. 赔偿限额的高低
    4. 保险责任范围的大小
    5. 免赔额（率）
    6. 是否参加当地的工伤保险
+ 保险费计算
    1. 按工资总额计收保费方式
    2. 按限额计收保费方式
    3. 按照工程造价计收保费方式
+ 司法管辖

### 第四节：案例介绍

## 第三章：产品责任保险

### 第一节：风险调查与评估

+ 投保人、被保险人的基本情况
+ 产品基本情况
+ 产品制造过程和风险控制情况
+ 产品质量认证
+ 产品销售情况
+ 产品损失及召回记录

### 第二节：风险控制

+ 被保险人：被保险人名称和地址；被保险人营业性质
+ 被保险产品
+ 销售区域与销售额
+ 赔偿限额
+ 免赔额
+ 保险费率的厘定
+ 索赔基础
+ 承保区域
+ 司法管辖
+ 附加条款
    1. 以索赔提出为基础条款
    2. 以事故发生为基础条款
    3. 索赔处理费用（适用自负额）包含在赔偿限额内。该条款只适用于出口产品责任险。
    4. 销售商扩展条款（不指定/指定）。国内产品责任险一般不扩展。
    5. 产品完工条款。该条款实际上是维修商扩展条款，仅极少数出口产品责任险业务会要求附加此条款
    6. 绝对石棉污染除外条款。建议在承保建筑材料等可能含有石棉成分的产品时附加。
    7. 指定公估人条款。国内产品责任险不需指定，出口产品责任险应指定公估人，以便 于赔案的有效处理。
    8. 铅污染除外条款。适用于承保铅酸蓄电池等产品
    9. 功效除外条款。适用于承保功能性产品，如药品、消防器材、粘合剂等
    10. 电磁波辐射及无线电波除外条款。适用于承保电子、电器产品
    11. 过敏症除外条款。适用于承保化妆品、药品等
    12. 基因改变除外条款。适用于承保食品、饮用水、化妆品、药品等。

### 第三节：承保方案

## 第四章：职业责任保险

### 第一节：建设工程设计责任保险

+ 风险调查与评估
    - 特别注意了解以下信息：
        1. 调查被保险人承保前有无接办重大建设工程设计任务，尤其是否有因设计质量低劣引起返工或因设计错误而造成工程重大质量事故的
        2. 被保险人所雇佣的设计人员的姓名及其工作经历、年限等。
        3. 投保设计院/所的营运及盈利能力状况
        4. 被保险人所雇佣的设计人员的技术水平、个人素质及职业道德状况
        5. 内部质量控制管理系统的有效程度
        6. 调查被保险人是否有被职业索赔的经历，是否曾经投保过职业责任保险，未投保的原因等
        7. 调查被保险人是否曾有前任被解职，解职的原因是因为疏忽、过错行为还是因为诈骗或其他恶意行为
        8. 要求附加的保险保障。
    - 建设工程设计风险主要包括：
        1. 法律风险
        2. 技术风险
        3. 人为风险
        4. 程序风险
+ 风险控制
    1. 被保险人资质
    2. 投保区域范围
    3. 年设计费收入
    4. 赔偿限额
    5. 设定每次事故索赔免赔额
    6. 被保险人经营时间以及参与大型项目或者类似项目的经验
    7. 被保险人以往赔付记录以及其他险种的赔付状况
    8. 需要附加或扩展的保险保障
    9. 其他管控措施
        - 小型单位，应请客户配合填写基本情况清单
        - 大型单位，可用承保当月的工资表代替本表，但出险时，设计人员必须在事故发生时的工资表中体现
        - 设计人员变动时，应及时通知保险人进行批改
+ 承保方案
    1. 使用的保单条款
    2. 明确的赔偿限额
    3. 每次事故索赔免赔额
    4. 保险费和费率
    5. 保险期间
    6. 追溯期。确定原则：
        - 一般情况下，第一年投保时，不给予追溯期；第二年续保时，追溯期追溯到第一年保险单的保险期限起始日；第三年续保时，追溯期仍追溯到第一年保险单的保险期限起始日，即连续投保，连续计算追溯期。投保人未续保一段时间后重新投保的，追溯期重新计算。对某些被保险人提出的第一年投保时给予一定追溯期的要求，保险人在进行充分风险评估的基础上，在被保险人补交一定保险费后，可以适当考虑。
        - 追溯期最长不能超过5年，追溯期较长的情况下可以适当加收保费，一般情况下，追溯期越长，费率越高。
    7. 承保区域范围
    8. 司法管辖权范围
    9. 需要附加或扩展的保险保障。

### 第二节：单项建设工程设计责任保险

+ 风险调查与评估
+ 风险控制
    1. 审核投保人有关信息
    2. 被保险人执业许可证编号
    3. 工程设计单位类型。一般分甲、乙、丙三级，资质级别从甲级开始递减。
    4. 工程项目的种类
    5. 工程地点
    6. 投保区域范围
    7. 保险期限
    8. 追溯起期
+ 承保方案
    1. 投保人/被保险人情况。
    2. 保险期限
    3. 承保区域范围
    4. 责任限额：每次事故责任限额与累计责任限额；每人伤亡责任限额
    5. 免赔额：低费率对应高免赔；高费率对应免赔额标准为每次事故人民币3万元或损失金额的10%，以高者为准；低费率对应的免赔额标准为每次事故人民币150万元或损失金额的10%，以高者为准。
    6. 保险费率：包括基准费率和调整费率
    7. 保险费
    8. 保险期间
    9. 追溯期
    10. 司法管辖权
    11. 特别约定

### 第三节：律师职业责任保险

+ 风险调查与评估
    1. 被保险人承保前有无接办重大法律事务，尤其是对有无重大败诉案件要了解清楚
    2. 被保险人完成业务的所有办公地点，被保险人合伙人的姓名及合作年限等。
    3. 投保事务所的营运及盈利能力状况，事务所的规模、业务类型、资信情况及业务复杂程度。
    4. 执业律师（专、兼职）的数量、技术水平、个人素质及职业道德状况。
    5. 内部质量控制管理系统的有效程度。
    6. 了解被保险人是否有被职业索赔的经历，是否曾经投保律师职业责任保险，未投保的原因，曾经投保的情况。
    7. 了解被保险人是否曾有前任被解职，解职的原因是因为疏忽、过错行为，还是因为诈骗或其他恶意行为。
    8. 要求附加的保险保障。
+ 风险控制
    1. 明确被保险人：有两种，律师事务所和执业律师。本保险只承保律师事务所及参与律师事务所工作的单个执业律师的职业责任保险。
    2. 触发机制。往往采取“期内索赔式”
    3. 免赔额
    4. 协议项下的责任
    5. 投保区域范围：中国境内（港澳台除外）
    6. 事务所类型：国资、合作或合伙等
    7. 赔偿限额及费率
    8. 每次索赔免赔额：一般为10万元或将免赔率定为每次损失金额的10%，以高者为准。
    9. 保险费：以被保险人预计当年业务收入为基数乘以适用的费率，在保险单到期后1个月内，被保险人应提供在保险单有效期内实际业务收入的有关账册，供保险人查验，根据实际收入金额，计算调整预收的保险费，多退少补。
    10. 追溯期
    11. 保险期限
    12. 以往事故情况：详细填写被保险人前五年责任事故情况
+ 承保方案
    1. 使用保单名称
    2. 被保险人的名称、营业处所地址、执业许可证号及事务所的类型
    3. 明确承保区域范围
    4. 确定赔偿限额、计算的方法及具体标准
    5. 规定免赔额的确定方法，并明确该部分为被保险人自负责任的金额
    6. 司法管辖权范围
    7. 费率的确定及保险费计算、缴付方式
    8. 需要附加的保障以及其他条件、加费幅度

### 第四节：注册会计师执业责任保险

+ 风险调查与评估
+ 风险控制
    1. 保险责任：注册会计师执业责任保险只承保条款列明的注册会计师承办的审计业务的责任，不包括非审计业务。注册会计师从事执业活动应当加入会计师事务所。
    2. 保险条款中涉及的利害关系人是指按照法律法规规定有权使用注册会计师审计报告的投资人、债权人。
    3. 事务所类型。分为有限责任会计师事务所和无限责任合伙会计师事务所两种。
    4. 赔偿限额。基本赔偿限额为每次100万元，累计200万元。
    5. 免赔额：每次索赔的最低免赔额为1万元
    6. 保险费和费率：以拥有注册会计师的人数为计收依据，因而未规定相应的费率，而只是规定了基本赔偿限额下的基本保费及增加赔偿限额后的基本保费倍数。
    7. 追溯期
    8. 保险期限：通常为一年，为防止道德风险发生，无特殊情况，保险人原则上不能同意投保短期保险的要求。
    9. 投保区域范围：中国境内（港澳台除外）
+ 承保方案

### 第五节：医疗责任保险

+ 风险调查与评估
    1. 医疗机构的合法合规性
    2. 医疗机构的经营类别、执业范围、经营规模以及所在地域。
    3. 医疗机构的营运状况和盈利能力
    4. 医疗机构中医务人员的数量、所在科室、技术水平、个人素质及职业道德状况，特别是部门以上管理人员的资历和水平等。
    5. 医疗机构的病床数量、病床利用率、住院人数、门诊量等情况
    6. 内部规章制度的建立和执行情况
    7. 以往医疗纠纷情况及其他赔付因素
    8. 医疗机构应对医疗事故的处理能力
+ 风险控制
    1. 客户选择
    2. 客户管理情况
    3. 赔偿限额：医疗机构须采用统一的每人赔偿限额和累计赔偿限额，不允许对不同科室设定不同的限额。原则上医疗责任保险每人赔偿限额一般在20万元左右，最高不超过50万元，累计赔偿限额不高于200万元，无累计赔偿限额的不能承保。
        1. 医疗责任赔偿限额
        2. 法律费用。为医疗责任赔偿限额的一定比例，免收保险费
        3. 精神损害每人赔偿限额。也是医疗责任赔偿限额的一定比例，但一般承保中都不会给予此项扩展。
    4. 投保方式。必须采用全部投保方式。不能只选择部分科室投保。
    5. 保险费。分为“分类计算”和“统扯计算”两种方式。
+ 承保方案
    - 主要包括：被保险人；赔偿限额；保险费[基本保险费、保险费调整]；保险期限；追溯期；免赔额；附加险

## 第五章：其他责任保险

### 第一节：物流责任保险

+ 风险调查与风险评估
    1. 被保险人资质及基本情况
    2. 核实物流业务收入
    3. 主要业务类型
    4. 自有车辆类型及车况
    5. 物流地域范围、运输路线
    6. 承运货物类型
    7. 承运司机资质及相关风险管控措施
    8. 仓储情况
    9. 运输方式
    10. 是否使用专业物流软件进行管理
    11. 是否已投保相关险种
+ 风险控制
    1. 年营业收入2000万元以下的新保业务，但续保条件较好或以往赔付率较低的业务可适当放宽要求。
    2. 物流企业的外包、转包业务
    3. 运输的货物是否包含高风险货物
    4. 跨国境的公路运输业务
    5. 未配备专业物流软件的物流企业，以及不接受保险人随时查验该系统的物流企业。除非逐单申报。
    6. 累计无限额的业务
    7. 露天仓储业务
    8. 管理水平低下，未制定安全运输管理制度、仓储管理制度，风险管控水平较低的物流企业
    9. 车况情况较差甚至超期运营的，或存在私自改装车辆、严重超载、违章经营、违规装卸现象的物流企业。

### 第二节：董事责任保险

+ 风险调查与评估
    1. 投保公司的基本情况
        1. 公司财务状况
        2. 公司所在行业状况和公司的行业地位
        3. 关联公司业务性质
        4. 投保公司近期股价表现
        5. 内控管理情况
    2. 董监事及高级管理人员个人情况
    3. 以往事故情况
    4. 其他险种赔付情况
    5. 其他可能影响风险的因素
+ 风险控制
    1. 限额控制：通常每次限额与累计限额是同一个限额，所以应严格控制累计责任限额。
    2. 免赔额控制。董责险一般免赔额为零；针对中国内地上市公司，公司补偿责任、公司证券责任免赔额需控制在5万人民币以上。针对香港上市公司，公司补偿责任，公司证券责任免赔额需控制在20万人民币以上。针对美国上市公司，公司补偿责任、公司证券责任免赔额需控制在50万美元以上。公司雇佣责任免赔可根据公司的具体风险灵活设置。
    3. 责任免除控制
        - 董事的故意行为
        - 董事的个人行为，即董事非以公司董事身份履行职务的行为
        - 公司内部纠纷
        - 非在保险期间对董事提出的索赔
        - 法律规定禁止承保的责任、违反公共政策（公序良俗）的责任
    4. 扩展责任/附加条款控制
    5. 追溯期控制
+ 承保方案
    1. 赔偿限额。针对非上市公司，累计赔偿限额通常在2000万人民币以内；针对国内上市公司，累计赔偿限额通常在5000万人民币以内；针对海外上市公司，累计赔偿限额通常在3000万美元以内
    2. 免赔额
    3. 保险费率
    4. 保险费：以赔偿限额为基础计收
    5. 保险期限
    6. 承保地区范围
    7. 司法管辖
    8. 追溯期
    9. 发现期

### 第三节：建筑工程质量保险

+ 风险调查与评估
    1. 风险调查与评估的准备工作
    2. 风险调查与评估
        1. 勘察工作失误，勘察布点不满足规范要求造成土层性质判断失误，引起基础方案与地基土不相适应，以及勘察取样和试验结果评价有误，造成地基不均匀沉降，引起影响结构安全的损伤。
        2. 设计错误或疏漏
        3. 施工过程中的质量控制不严
        4. 工程完工后，有的开发商和建设单位已不复存在，使建筑工程维修等难以实施
        5. 建筑工程竣工后一些内在缺陷还没有完全显露出来
        6. 明确和确定建筑工程质量检查控制机构
        7. 明晰建筑工程质量检查控制机构的聘请方式
        8. 确定建筑工程质量检查控制机构的工作内容
+ 风险控制
    1. 投保人。我国对房地产开发企业实行资质管理制度，除具备企业法人一执照外，还要具备相应的资质证书。资质分为1--4共四个等级，其中一级最高，四级最低。应尽量选择资质较高、信誉较好的房地产开发企业。
    2. 被保险人。
    3. 保险标的。为住宅商品房及写字楼，其他建筑工程不在本保险承保范围内。
    4. 投保时间。保险人一般只接受投保人在工程开工前的投保申请。
    5. 工程质量检查控制机构。投保人填写投保单的同时，必须与保险人指定的质量检查机构签订建筑工程质量检查协议。
    6. 预计总保险金额
    7. 检查费用。不包含在保险费中，由投保人另行交付，由保险公司通过代收代付的方式支付给检查机构。
+ 承保方案
    1. 使用的保单条款
    2. 赔偿限额
    3. 每次事故索赔免赔额
    4. 保险费率
    5. 审核预付保险费
    6. 签发保险单
    7. 合同生效
        - 竣工验收合格证书
        - 工程施工承包单位出具的工程质量保修书，建筑物使用说明书
        - 质量检查机构在竣工验收时对上述建筑项目出具的建筑工程竣工验收质量评估报告和在竣工验收合格满1年后对上述建筑工程项目出具的建筑工程质量检查通过报告。
        - 上述建筑工程项目在质量检查机构检查通过时的单位建筑面积平均销售价格和实际总建筑面积。
        * 上述建筑工程项目竣工验收合格满1年后未能通过质量检查机构检查的，保险人不再向投保人签发正式保险单，在扣除预付保险费10%的手续费后，将剩余部分预付保险费退还投保人，保险合同终止。
        * 投保人在向社会公众应投保建筑工程质量保险事宜进行宣传或报道时，其内容必须事先得到保险公司的书面认可，并交付保险公司标志使用费用30万元人民币或预付保险费的30%，二者以高者为准。
    8. 实际保险金额与保险费
    9. 特别约定。将相关质量检查机构出具的保留意见（如有）的内容记入特别约定，作为除外责任。
    10. 保险期间。建筑工程质量保险主险的保险期间为10年，自建筑物竣工验收合格满1年后开始。如选择投保附加渗漏保险，附加险的保险期限为5年。

# 第五篇：理赔篇

## 第一章：理赔调查

### 第一节：理赔调查准备
