## 什么是索赔

当客户的投保标的物损坏或者丢失时，可以通过递交索赔申请来向其保险人寻求赔偿。索赔申请既可以是口头的（即电话申请），也可以是书面的，即填写并向保险人递交索赔申请表。

## 索赔处理过程

收集索赔的有关资料；对资料进行分析；对索赔展开调查；记录详细情况/数据；确定完成理赔；

+ 小额索赔  
小额索赔特点是处理过程简单，理赔耗时也相对较短。具有此特点的索赔包括玻璃损坏、器件熔化、食品腐败、档风玻璃损坏等。
+ 较大数额和较为复杂的索赔  
较大数额和较为复杂的索赔将需要更长的处理时间，通常受以下因素影响：  
整理证据所需的时间；修复损坏所需的时间；物品及材料的供应状况；维修人员的时间安排；  
工伤赔偿保险和个人事故/疾病保险等的索赔处理，所需要的时间要比一般保险长得多。

## 接受索赔

+ 提供紧急援助  
援助服务一般都是通过电话服务中心提供。紧急援助可以缓解客户的紧张情绪，使他们确信自己的需要会得到满足。紧急援助可能包括：  
指导提出索赔所需要的步骤；为因车辆损坏而陷入困境的客户安排交通；在情况严重时安排医疗援助、治疗或安排住院；为损失地点安排紧急服务，目的是进行紧急修复，以避免建筑物进一步受损。

## 澳大利亚的立法与索赔处理

保险合同法1984；商业行业法1974；澳大利亚证券委员会法1989；金融服务改革法2001；隐私权修正（私营产业）法2000；  
普通保险行为准则；

+ 保险合同法：不适用于工伤赔偿保险、强制第三方责任保险、海事保险和未经该法特别指明的其他任何保险。  
未如实告知和虚假陈述，保险人可以不进行赔付，并且在扣除过去所赔付的金额之后，退还剩余的所有保费。  
比例分担条款/共同保险条款：不生效条件：未做出明确的解释；投保价值达到80%以上。  
如果有两份以上的保单保障，接受索赔的公司必须支付100%的索赔款，然后，支付索赔款的保险公司再向其他保险公司寻求赔偿。  
预先存在的缺点或瑕疵：可以拒绝赔付，但被保险人预先不知道或正常情况下也不可能知道的除外。  
预先存在的疾病或残障  
第三方向保险人追偿的权利  
保险人在特定情况下无权拒绝理赔  
欺诈性索赔：保险人有权拒付全部索赔金额，且无须退还保费。  
索赔利息  
普通保险合同的撤销  
代位追偿针对家属等人的权利：保险人无权向未保险但有过失责任的下列第三方人士追偿：被保险人的家庭成员或者被保险人的朋友。  
代位追偿针对雇员的权利：但雇员严重或明知故犯的错误导致的损失除外  
适用于代位追偿金额的权利

+ 普通保险行为准则  
是保险业尽可能提供最佳服务以及使客户和保险人之间关系更为融洽的承诺。  
适用险种：私人机动车辆，室内财产，消费者信贷，个人和家庭财产，家居建筑物，疾病与事故，旅行。  
1995版《准则》有如下规定：  
索赔申请表和索赔程序等文件必须采用用户容易掌握的形式，以简单明了的语言撰写，并向公众免费提供；  
索赔必须立即得到处理；  
须随时就索赔的进展状况与客户保持联系；  
必须提供有关内部和外部纠纷解决程序的信息。  
2005版《准则》适用于所有普通保险产品，除：工伤赔偿保险；海事保险；医疗赔偿保险；第三者责任强制性保险。  
《准则》不适用于由人寿保险公司或注册健康保险公司签发的人寿保险和健康保险。  
概括地说，《准则》有如下规定：
- 如果收到了所有必要文件而且无须进行进一步调查，那么保险公司必须在接到索赔要求10日内决定接受还是拒绝索赔，并将该决定告知被保险人； 
- 处理索赔须适用统一的标准，以确保索赔处理过程的公正、及时和透明； 
- 需要更多资料、评估或调查的索赔须适用附加标准； 
- 如果某些性质的索赔在规定时限内无法处理，则可以代之以其他时限； 
- 员工和服务提供者处理索赔适用统一标准； 
- 特定情况下可能适用经济困难规则，使申请人能以更快的速度获得赔款； 
- 特定情况下可能就第三方追偿适用经济困难规则； 
- 如果保险人直接选定并授权修理者，那么他们必须对工艺和材料的质量负责，并负责处理与修理者行为有关的投诉。

## 索赔

保险人和被保险人相互有义务遵守最大诚信原则

+ 如实告知义务--澳大利亚  
被保险人无须如实告知：能减轻风险的任何因素；常识性的信息；保险人知道、或者在正常情况下应该知道的信息。

如果投保人/被保险人未能遵守如实告知义务，《保险合同法》允许保险人采取特定的救济措施，包括允许保险人全额拒绝索赔、或者减去与投保人/被保险人在订立合同时的虚假陈述或未如实告知相对应的那部分索赔金额。

## 接到索赔

向客户说明的具体内容：
- 索赔过程，包括是否指定理算师和索赔评估人；
- 索赔是否在保单的保障范围之内；
- 预计处理索赔的时间；
- 客户是否需要承担任何免赔额，以及该免赔额必须何时支付；
- 提出索赔是否划算；
- 保险人可以立即为客户做出哪些安排--如：紧急援助；
- 首选供应商的报价，或者，认可客户所选的供应商；
- 客户需要提供哪些文件--如：警方报告，驾驶执照，和所有权证明等。

+ 被保险人的义务
- 发出索赔通知（书面或口头）
- 确保在保单条文所规定的期限内发出通知
- 采取一切可能措施以降低风险
- 提供损失证明，如：收据，估价发票，账目等
- 任何时候都遵守最大诚信原则

## 估算索赔额：这一过程被称为设立准备金

准备金是预留出来支付索赔的资金。准备金数额是保险人借以衡量和公布其财务状况的关键工具。

+ 准备金政策

- 在设立准备金数额时的一致性
- 能够迅速、准确地确定可能的索赔费用
- 得到额外资料或发生重要进展时立即对准备金进行审核
- 必要时立即对准备金进行调整

保险人通常为以下事宜的处理设定特别的规则：
- 追偿行动/诉讼

# 第3章：调查索赔

## 保单诠释

+ 是否属于承保范围
+ 保单条文与成本估算
- 未足额保险
- 超额保险：如果某些财产超额保险，保险人在理赔时就需要支付过多的赔偿金
- 自负额或者特殊条款
- 再保险
- 代位追偿权：保险人只有在被保险人完全得到赔付后才享有代位追偿权
- 专业人员：保险人可能需要指定的两类专业人员是理算师或损失评估人

## 是否需要进一步调查

在决定索赔结果时，需要考虑的因素：
- 如果索赔将被接受，有哪些理赔选择
- 如果需要额外资料或建议，需要采取哪些步骤来进行深入调查和/或指定专业人员
- 如果索赔将被拒绝--拒绝的程序是什么

不应只在发生大额或不寻常的损失时才决定调查索赔，在索赔的初步评估阶段就要做出合理的判断。  
专业的索赔管理会影响贵组织的声誉，而另一方面，也需要考虑欺诈的可能性。

## 处理欺诈

涉嫌欺诈；被保险人欺诈；第三方欺诈；与供应商有关的欺诈；缺乏真实性的其他任何索赔。

+ 何谓欺诈标志  
保险人必须证明存在三种因素方可拒绝赔付欺诈性索赔：
- 欺诈是故意的
- 被保险人必须有欺诈动机
- 有机会实施欺诈

澳大利亚的保险调查专员服务（IOS）  
新西兰的保险与储蓄调查专员计划（ISO）

## 专业人员的作用

+ 理算师与评估人：是作为独立的一方，就索赔的处理问题向保险人提出建议。但是，除非另有授权，他们无权理赔或者拒绝赔偿。

+ 其他专业人员：调查员，会计师，工程师，估价师，火灾起因与起源专家，摄影师，其他人员

+ 资料汇报指引  
理赔人需要在调查开始之前或者在调查期间向专业人员提供完整和准确的资料，在对索赔做出决定之前认真领会专业人员的报告与建议。

+ 调查报告的验证  
申请赔偿的物品在所声称的损失发生之前就已确实存在，而且也是被保险人所拥有。保险人所需要的最觉的证据是投保物品的收据或者发票的副本。

# 第4章：最终完成理赔

最终完成理赔是索赔过程中的最后一步。其中包括通知客户理赔结果、完成估算和进行赔付（如果接受索赔的话）。在整个过程中，确保客户了解情况非常重要。

## 确保客户经常接到关于其索赔进展情况的通知

+ 澳大利亚

保险人有责任：  
迅速有效地处理索赔；随时告知被保险人索赔处理的进展情况；决定是否接受索赔后立即通知被保险人

+ 新西兰

## 接受或拒绝索赔

+ 完成下列步骤后索赔即告完成：  
完成对保单和估算费用的审核与分析；  
搜集并查阅专业人员提交的报告；  
确定该项索赔是否在您的权限范围内；

一旦做出决定后便马上通知客户是最重要的。

+ 拒绝索赔

保单过期；索赔内容属于保单的免赔范围；适用除外条款；被保险人未能如实告知；存在欺诈行为；索赔财产不在保障范围内。

一旦决定拒绝索赔，您需要向被保险人发出通知，明确解释拒绝索赔的理由。  
明确给出拒绝的原因，必须向客户提供有关纠纷解决程序的信息。

在这些理由中不需透露：
- 第三方所提供的机密资料或者第三者的身份
- 受法律专业特权管辖的资料
- 会妨碍保险人就有关索赔作出任何进一步调查或争辩的资料

+ 接受索赔

如果索赔会被接受，您需要向被保险人或其保险中间人发出通知，包括：
- 按照保单规定索赔有效
- 被保险人是否需要支付免赔额
- 赔付方式
- 完成理赔所需的进一步措施

保险人支付索赔时有四种选择：  
为物品支付现款；转换损坏或丢失的物品；修理损坏的物品；复原受损的财产。

+ 现金支付：是支付索赔最显而易见的方式，但最不为保险人所喜爱。

## 消费税

+ 消费税--澳大利亚
+ 投入税返还（ITC）和索赔支付--澳大利亚

## 恢复投保总额

+ 海事保险：海事保险中的部分损失赔偿不会降低此后的保障数额。因此，连续的部分损失赔付可能使赔偿总额超过投保总额。
+ 火灾保险。保险人的赔付总额不能超过投保总额。
+ 其他种类保险。恢复投保总额不适用于人寿保险，因为在一次索赔支付后，合同会自动终止。
- 机动车辆条款的“自损”条款，同一保单年度内赔付多次修理不减少投保总额。多数家居住宅和室内财产保险也在索赔后继续提供全额保障。

+ 损坏--澳大利亚  
在过失诉讼中，存在“损坏”赔付。

## 最终确定估算额

+ 赔付：常见的支付方式是支票结算或者电子转账。  
客户损失报价；专业人员服务费用；第三方当事人索赔或者追偿损失；公共责任或者个人人身伤害索赔。

## 最后核查

+ 索赔决策应当按照保单条文、法规、组织运营程序和行为准则所要求的方式明确通知相关各方。 
+ 在可能的情况下，应当以能够反应决策基础以及所有相关证据/资料的方式来记录索赔决策。 
+ 索赔文件应当送交再保险人/共同保险人。
