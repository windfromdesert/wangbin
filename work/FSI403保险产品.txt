保险产品

# 第一章：保单与保险产品

保单通常是保险人的产品，是保险合同的一种证明

## 不同种类的保单

+ 法定保险

提供法定保险的保险人：不能决定应提供哪些保障；不能拒绝为提出法定保障要求的客户提供保险。

+ 规定保险

引入规定保险的目的是：保护那些将保险从一家保险公司转到另一家保险公司的客户。  
澳大利亚有六类保险被界定为规定合同：私人机动车辆保险（包括2吨以下的车辆）；家居建筑物保险；家居室内财产保险；患病与意外事故保险；消费者信贷保险；旅游保险；  
新西兰的准法定类保险：伤害预防、康复和赔偿法；地震保险委员会（EQC）

+ 商业或企业保险：通常被用来描述中小企业（SME）的各种保险需求。中小企业包括零售业、商业和工业领域。保单会以“组合”的形式或者小册子的形式推广。

+ 人寿保险：为因任何原因而引致的身故提供保障（只有几种情况除外），这类保单也可以是“终身”保险合同，这种合同既可以提供身故保障，又具有现金投资性质。

+ 消费者信贷保险：当出现身故、受伤或患病、失业的情况时，消费者信贷保单可以为客户提供信用或者个人贷款债务保障。这种保障仅限于个人，而且保障数额仅限于合同中所规定的数额。这种保险通常是作为贷款组合的一部分而办理的，然而，它不具备强制性质。

+ 收入保障

在发生意外事故或者患病的情况下，收入保障保险可以为被保险人提供部分个人收入保障。受益期可以与多种不同的免赔额（表述为时间段时被称为等待期）结合起来，以适应客户的不同需要。一般情况下，保单可得到续保保证。

人寿保险公司和普通保险公司都可以提供收入保障保险。通常情况下，人寿保险公司可续保期一般可以持续到正常的退休年龄；而普通保险公司在提供收入保障时会带有更多的限制条件，如收入保障可以被限制在最多两年之内。

+ 旅游保险

能为客户提供下列保障：交通费损失；特定旅程中的住宿费损失；特定旅程中个人财产损失或者损坏；特定旅程中客户患病时的医疗费用；个人意外保险；遣送费用；

多数旅游保单也为个人责任提供保障。通常情况下，旅游保单可以直接签发给旅行者个人，也可以签发给公司团体。

+ 个人责任保险：家居保险中会自动包括法律责任部分，这种保障仅限于在建筑物范围内所发生的意外事故；在室内财产投保之后，法律责任保障还会扩展到个人法律责任保障，该险种适合于澳大利亚或新西兰境内任何相关地点--有时这种保障可以推广到全球。室内财产保障还包括客户在租住建筑物中所承担的法律责任。这种保障专为家庭市场设计。

+ 公共责任保险：保障与客户的特定业务活动/贸易活动相关联、可能会造成人身伤害和/或财产损坏的潜在法律责任的领域。

+ 财产保险

火灾与危险保险；火灾与扩展危险保险；玻璃破碎保险；建筑物所有人/房东保险；机械故障（工程）保险；现金保险；盗窃保险；建筑与安装保险；行业特殊风险（ISR）保险；电脑保险[电脑病毒造成的损失或者损坏通常属于除外责任，在新西兰，病毒风险已被纳入标准保障的范围]；业务中断（也被称作是间接损失或者利润损失）保险；

+ 其他责任保险

社团责任保险；承运人责任保险；承租人责任保险--船舶承租人；承租人责任保险--航空责任保险；诽谤保险；董事和经理人责任保险；电子行业保险--错误和疏漏；雇用责任保险；普通责任和产品责任保险；污染/环境损害责任保险；专业责任损失补偿保险；受托人责任保险；合同工程责任保险；伞护式责任保险[对“潜在险种”里的各种缺口进行保险，可以设定赔偿上限]；机动车辆第三者责任保险（仅限于新西兰）

+ 机动车辆保险：强制性第三者人身伤害保险；第三者财产保险；综合险[对客户自有车辆由于任何原因造成的丢失、被盗或损坏进行赔付]；

+ 海上保险

《海上保险法1909》适用于商业海上保险；《保险合同法1984》只适用于自用船只的海上保险，也就是说用于娱乐目的的船只保险；  
自用船只：由一名或多名个人所有，用于娱乐或体育活动，不用于营利目的。

船舶保险；责任（保障与补偿）保险；娱乐性船舶保险；海上货物保险；

+ 航空保险：飞机保险；责任保险；飞机场所有人/机库负责人保险；承租人责任保险；

+ 保证书：是担保人给权利人的一种保证，以此保证委托人会履行与权利人缔结合同的义务。保证书不是保单。

+ 灾害保险

电脑犯罪保险；公务旅行保险；农作物保险；忠实保险；林业保险；绑架与赎金保险；牲畜保险；个人意外事故/患病保险；

+ 其他杂类保险

商业法律费用保险；信用保险；董事法律费用保险；出口信贷保险；伪造股票转移保险；知识产权保险；恶意产品损害保险；多风险/普通财产保险（也时也称为“全风险保险”）；产品保证保险；产品召回保险；收购抵御补偿保险；税务审计保险；

## 能否保险要看哪些因素？

承保损失具有随机性和偶然性；可能发生很多类似的情况；不会出现灾难性损失；保险人必须能够对损失机率进行较为准确的计算；必须制定出可行的保费，以确保产品得以延续，损失必须可以用货币衡量；保险不能违反公共政策；

+ 风险评估

确定承保风险的条款和条件；将风险评估的结果告诉客户；评估客户的反应；  
风险评估限定在：客户评估（不论是个人还是企业组织）和组合业务评估。

风险识别就是辩明保险人可能要面对的种种潜在的风险因素或者损失。  
个人或者企业组织评估  
风险因素：个人或者企业组织的活动；自然因素；可能增大机率的与个人性格和道德水平相关的道德因素；因为有保险，便造成了损失，该类损失与疏忽大意等精神状态的因素相关。  
任何一种风险识别方法都不可能洞察某个企业组织或者个人所面临的所有风险。企业组织和个人的生活环境是动态的，因此，风险识别是一个持续不断的过程。  
过往记录：不仅可以证明潜在损失的来源，而且还可以证明损失的概率和严重程度。  
保险业务评估  
风险评估相关的风险因素：企业运作的社会、法律和政治环境；天气变化和自然灾害所造成的风险；保费数额是否足够、接受原则、合同条款和监控体系等。  
风险评估：评估时，风险因素将被量化为风险度单位。  
风险程度：是风险的一种衡量手段。

## 保单的结构

标题--载明相关保险的种类；  
述辞或者导言--是指客户提供的投保申请书或者资料，这些资料对保险人的重要性，确认合同的各方当事人，客户支付保费的承诺；  
实施或者保险条款--保险人的义务，投保事件，保险期间，条件和除外条款适用的事实情况，保险人的责任限额；  
一览表--保单号码，投保数额，投保财产的地址，其他当事人[如承受抵押人]，保险期间，客户详情，投保财产，保费，保险中间人的名称[如果适用]，自负额；  
除外条款与条件--除外条款，条件；  
背书批注--记录对保险人标准合同条款的变更；

结构的变更：可能会使用不同的术语来表示各个不同的组成部分，而其先后顺序也可能会有所不同。

磋商协议

临时保单：临时保险与正式保单具有同等的约束力，正式保单适用的法律原则对临时保单同样适用。

通用定义--通常会列出适用于整个保险组合的关键词或词组，并会对其界定；  
组合保险的各个部分--保障的第一部分[定义；保障；投保事件；额外利益；理赔依据；共同保险或者不足额保险；条件；除外条款]；

# 第3章：保险文件

承保文件；索赔和变更文件；续保文件；保险中间人文件

## 保险文件

+ 承保文件

临时保险合同（暂保单）--保险人准备提供的保险合同，提供临时性的保险保障，并且将被另一份保险合同所取代或者接替，无论该合同文件是否有书面或一般被称为暂保单/临时保单的文件作证明；  
投保申请书；  
保单册子--这是一份预先印制好的保单文件，上面载有保单条文、《保险合同法》所规定的如实告知条款、各种不同的普通保单条件和客户需要依据保单进行索赔时需要遵循的程序；  
保单一览表--绝大多数保单都是预先印制好的表格或者小册子，然后再加入个别客户的有关细节。当保单详情被输入电脑中的保单主文件之后，电脑将会编制一份保单一览表并附加到保单上；  
调查报告--实地勘察、编制报告、拍摄照片、指出潜在的隐患并提出建议；  
声明--是一种陈述，经申请人签名并声明申请人所提供的资料是真实的，无保留的；另外如果保费是根据一些估计数字而进行的话，可能需要另一种声明，对下一个续保期间的工资总额或营业额的估计，保险期满之后需向保险人声明实际数额，保险人将对保费做出调整。

+ 索赔与变更

与索赔相关的最重要的文件是索赔申请书，对于数额较大的索赔还需要提供其他的各种报告/报表；

在保单中对变更进行记录的主要方式是使用背书批注，有些变故可能需要保险人签发一份保险凭证。

索赔申请书要求客户以书面形式提供任何索赔的详细情况，以及向保险人提供可能需要的所有资料和文件；

背书批注用于：在保单一览表中详细注明变更情况；对基本保单中所提供的保障范围进行扩展、限制或者修改；删除某项除外条款。背书批注可使用电脑编制，附在保单一览表上，也可以预先印制在带有电脑生成编号的保单上。

保险凭证：对投保财产具有经济利益的当事人可能需要出具保险凭证，例如：银行。

+ 续保

续保通知书：保险人必须在可以办理续保手续的保险期满日前最少14天向客户发出书面续保通知书。包含：保障满期的日期和时间；保险人是否准备与客户协商办理续保手续或者扩展保障范围；

续保证明：保险公司签发续保证明，确认保单已经办理了续保手续。在新西兰，签发收条起到的作用大致相同。

+ 保险中间人

报价单：是一种简明投保申请书，上面注明了风险的最重要特征的详情。它将会被呈送给多家保险人，以获得保险人的报价。保险中间人选中了最有利的条款，得到客户授权按照这些条款投保时，报价单开始称为交付单。

交付单：载明了保费、自负额/免赔额和其他条文。保险人签署之后，上面会载明保险人接受该风险的百分比，如果接受承保的百分比低于100%，那么，交付单可以呈送给其他愿意依照主要保险人条款接受承保业务的其他保险人。其他保险人，一般被称为共同保险人，也需要签字关注明他们所接受的百分比。

## 各种保险文件之间的关系

源文件是投保申请书，投保申请书是保险人与被保险人之间订阅合同的基础。  
临时保险合同在客户正式填写投保申请书并将其返还给保险人之前，将为客户提供临时保障。  
接受投保申请后，保险人将确定保费。  
客户支付保费，而在投保申请书中客户也提供了详细情况，通常所有这些情况都被记录到保险人的电脑风险主文件中进行存档。电脑将编制一览表，一览表将被附在预先印制的保单册子上，之后一并送交给被保险人。  
背书批注与保险合同有关，因为背书批注是被用来对基本保单或者保单一览表做出变更的。  
续保通知书和续保证明是对保单续保的邀请或者确认。  
如果没有填写最初的投保申请、支付保费、收到带有保障条款与条件的保单和一览表，索赔申请书也就没有任何必要了。

## 不同类型的保险文件

为特定类型的保险选择了相应的投保申请书之后，随后签发的所有其他文件都将根据有关的保险类型发出，有关的文件也会体现这类保险性质。

# 第4章：了解最新的知识

## 产品知识的重要性

您必须了解：贵组织产品和服务的变化；保险业当前的动态；可能影响保险产品的法律变化。

+ 谁来承担责任

《普通保险行为准则2005》要求保险人保证对所有员工或者受权代表就其提供建议的产品进行充分的培训；还要求保险人确保其代理人和雇员不就没有接受过全面培训的产品为客户提供建议。

《澳大利亚证券与投资委员会（ASIC）政策声明146》规定了向零售客户提供金融产品建议者所需接受的最低限度的培训。

+ 组织指引和程序

组织指引：这些指引都是比较宽泛的，高层次的业务指引，可能用图表来表示，比如：流程图。

组织程序：组织程序是较低层次的、逐步的指引。

## 电脑在保险业的应用

+ 电脑系统

硬件；软件；电脑联网；局域网（LAN）；广域网（WAN）；互联网；

## 新兴技术

+ 电子资产

威胁：停电或者电流峰值过高；硬件故障；电脑病毒；大量垃圾邮件；员工、黑客或其他人的电脑欺诈；竞争对手、员工或其他人的恶意破坏。  
风险：敏感资料失去安全性[软件源代码；国防资料；银行账户]；网站和网络等重要资产无法使用；隐私损失；第三方提出起诉。  
损失：无形损失，损失数额可能十分巨大，这种损失也很难估计。

+ 电脑犯罪与法律

电脑犯罪包括：盗窃知识产权和数据库资料；盗窃信用卡资料；发送电脑病毒，破坏资料或者造成混乱；电脑欺诈和盗用客户信用卡账户。  
在有些国家，编写电脑病毒和实施黑客行为不算犯罪。  
病毒或者黑客袭击可以对世界各地的组织造成恶意的损失。

为了防范与虚拟环境操作相关的风险，保险人要做到：制定入侵监控策略；雇请专业人员分析病毒活动；审查在线流量，预测下一次袭击可能来自何方。

+ 在澳大利亚和新西兰，这方面的情况是怎样的？  
电脑保险都限定为：电脑设备的实体损失或者损坏或者故障和由于电脑硬件的实体损坏或者损失而造成的重建数据媒体的成本。  
电脑黑客恶意行为所造成的数据损失或者损坏不属于保障范围。

## 互联网在金融领域中的应用

+ 益处和局限性  
益处：保险人可以外包保险过程中的专项业务；保险人可以在线提供产品资料、接受投保申请、接受保费的支付并处理索赔，从而加强了现有的沟通渠道--例如电话服务中心，保险中间人等；可以快捷地更新产品和业务程序，而且成本低廉；电子商务可以降低产品营销成本、会计成本、配送成本和理财成本；电子商务可以降低进入市场的壁垒，因而增加保险业的竞争压力；客户可以方便地进行产品比较，以此做出更明智的选择。

局限性：企业和公众对网站的安全性缺乏信心，特别是对信用卡交易和传送数据的安全性心存疑虑；索赔仍然需要依赖传统的方法，要求提供原件书面文件证明；结构变化和员工培训的费用；大家仍然习惯于与人打交道，所以试图通过互联网开展全部保险业务的公司可能会面临与客户产生疏离的情况；并非所有的保险产品都同样适合通过互联网销售方式[复杂的保险产品，家庭类保单较适合在线销售]；
