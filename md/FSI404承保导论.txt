# 第1章：承保原理

## 何谓风险：

+ 风险的分类

动态风险与静态风险  
动态风险是指那些与经济情况的变化相关的风险。  
即使经济情况没有变化也会发生的损失被称作是静态风险。

基本风险与特殊风险  
基本风险为非个人性，它涉及大部分的社区公众。一般由政府出面处理。  
特殊风险自始至终均具有个人性。

纯粹风险与投机风险  
大多数情况下，纯粹风险可以承保，投机风险则不可以。

## 通过保险转移风险

## 可承保风险

损失必须具有偶然性（不可预见性）；损失不能过多；损失不能有悖于公共利益；必须存在足够的数目；损失几率必须能够评估；应该可以有金钱（财务）损失。

+ 不可保风险

违背公共利益；不能承保正常维护保养类风险；不能承保保险公司无能力承担的风险；不可计量的风险；

## 何谓承保

承保是保险业务中的技术性环节。它主要涉及接受或者拒绝风险，以及在风险可以被接受的情况下，确定承保适用的条款、条件和保费的决策。

+ 承保人的职责

权限级别

+ 承保的职能

个别风险承保  
组合风险承保

## 承保过程

作为承保人，您的主要职责有五项：  
选择风险；确定保单的条款与条件；确定风险的类别与费率；将数据输入公司数据库；在某些情况下，承保人需要确定有多少风险会由自己公司保留，以及需要进行再保险的数额。

+ 风险选择：是承保工作中最重要的一环。工作目标是实现组合保险业务的盈利。

风险选择工具  
贵组织的风险接受指引；向更有经验的承保同事请教；职业与行业方面的知识；现代工业与技术发展方面的知识；调查员的风险核查报告；最佳承保指引；个人常识与经验。

组织指引中还将包括：特批险种清单；拒保险种清单。

+ 保单章程（条款与条件）

+ 分类和费率

确定保险费率：确定保险费率是一个十分复杂、专业性很强的领域。  
资深承保人与精算师通力合作，根据索赔经验和每一个或一类风险所潜在的隐患，才可以计算出某种风险的保费。  
能够反映风险隐患的程度、满足索赔需要的保费被称作是“纯粹风险”保费，之后，再加上相应的费用、佣金和利润。  
保费通常包括：索赔成本；保单或者组合保险业务的管理成本（包括销售费用）；准备金、意外费用和利润差额。

+ 数据输入：准确、完整、及时

## 再保险

保险人将自己所承保的部分风险分给另一名保险人并向其支付保费。如果出现索赔情况，原保险人和再保险人将依据彼此之间的协议共同承担赔付。

保险人所承保的风险数额，将会计算：自留风险是多少？通过选择再保险转移了多少风险？

+ 为什么要进行再保险

再保险的好处  
提高对风险的抵御能力；抵御灾难性风险；提高财务稳定性。

+ 再保险的种类：选择再保险；固定再保险合同；共同再保险

换兑佣金：再保险人向原保险人支付佣金，被称作是“换兑佣金”，通常情况下，佣金将在原保险人向再保险人支付的保费中扣除。

固定再保险合同  
成数分保合同；溢额分保合同；超额损失分保合同；终止损失分保合同；预约分保合同；  
预约分保合同与选择再保险合同的区别在于，再保险人必须接受合同中所规定的所有风险。

共同再保险主要适用于灾难性或危害性高的风险。保险人可选择共同分担：所有的赔偿金额和所有的损失；超出规定自留部分的溢额；约定的损失超额部分。

# 第2章：风险的评估与选择

## 风险资料

除承保环节外，风险资料来自若干方面：  
组织的风险接受标准；您对不同种类隐患的理解；投保人/被保险人；以往的保险记录；独立的调查与核查；《最佳承保指引》

## 隐患的种类

+ 有形隐患  
财产保险：  
有形隐患与房屋的状况以及人们所从事的活动有关。  
房屋的结构；房屋的用途；房屋的保护措施；房屋的地理位置。  
责任风险：  
被保险人的活动；被保险人房屋；被保险人为了最大限度地减小损失所采取的预防措施；业务规模。

+ 道德隐患

取决于：被保险人的性格、品质和声誉；被保险人雇员或者同事的秉性；保险标的物，包括其是否容易受到时尚或者技术等因素变化的不良影响；被保险人管理层的责任心与效率。

道德隐患的例子：有贪污记录的员工；客户曾经被拒保，或者曾经有过犯罪记录；  
道德水准不高会增加风险中的不确定因素，增加保费数额或者特殊条款通常都无济于事，承保人不会承保存在道德隐患的风险。  
以往的损失令人怀疑；欺诈并虚报索赔数额；企业的经营行为有问题；有不良财务状况记录或存在明显的现金流动问题。

+ 理念隐患：理念隐患是由疏忽或者漠不关心造成的。

+ 法律隐患

## 风险的种类

+ 高于标准的风险
+ 低于标准的风险  
与“低于标准的风险”相适合的自负额/免赔额；限制性保障；改良风险；在再保险人认为风险可以接受的情况下进行再保险。

## 被保险人提供的资料

报价信息单；投保单；风险调查与其他调查。

# 第3章：投保单

## 投保单

投保单含有被保险人必须回答的若干问题。

+ 姓名/名称
+ 其他有利益关系的人
+ 地址
+ 风险的地理位置
+ 投保人的业务/职业
+ 年龄
+ 保险记录：先前办理过的保险；其他现有保险；索赔记录；拒保。  
拒保决定与投保人或个别风险无关的可以不必如实告知。
+ 个人记录  
任何有关盗窃、纵火、欺诈、超速驾驶或者酒后驾驶等方面的指控或者定罪；过去任何恶意伤及他人人身或者财产的事故；涉及毒品或者欺诈的犯罪；任何刑事犯罪指控或者定罪；任何破产、清算和债务判决记录等。
+ 保险标的物
+ 投保数额：超额保险；未足额保险；
+ 声明
+ 签名和日期

# 第4章：处理投保申请

## 保单的存续周期

续保时有必要进行审查，需要有定期的审查制度。

## 处理新的投保申请

接到报价请求/投保申请；分析风险；签发临时保单/临时保险合同；接受或者拒绝投保申请（或者提出任何反要约）；收取保费。

+ 分析风险
+ 作出决定
+ 保险索赔登记--新西兰
+ 签发临时保单：目的在于提供临时性保险保障及将被另外的一份保险合同所取代或者接替。保险人可以随时撤销某份临时保险合同。  
保险人不必就下列事项通知投保人/被保险人  
从适用规定合同保障转向适用临时合同保障；非正常条款；比例分摊条款；影响代位追偿权利的合同。
+ 收取保费  
全额支付；分期支付（常见于个人保险业务）；直接借记；信用卡支付；通过电话或者互联网支付；支票支付。
+ 登记保单

## 处理变更

搜集保单资料，并按照公司程序准确及时地进行记录；审核承保人的条款和条件；审核保单变更，确保其合乎立法和公司规定；在符合立法和公司规定的前提下，选择接受保单；按操作程序处理保单变更资料；按照操作程序检查对保单的变更是否准确、有没有疏漏；准确无误地升级资料系统；按照操作程序及时向客户发出变更文件。按照操作程序和法律要求处理或退还所有收到的款项；按照操作程序将相关文件准确存档；通知客户有哪些新增条款与变更前的保单不符。

+ 何谓背书批注？
+ 背书批注/条款的种类：背书批注在新西兰被称为“条款”。  
在澳大利亚，一般不允许保险人在保单有效期间对保单做出不利于被保险人的变更。保险人不能因为投保人被保险人提出索赔而中途增加保费数额。
+ 背书批注的起草：背书批注与保单文件中的语义，背书批注内容的效力较大。

## 处理续保

续保通知书中必须提醒投保人/被保险人履行如实告知义务。如果有任何情况改变而会影响保单的，一定要向您提供有关的资料。

+ 续保与《保险合同法》--澳大利亚  
保险人必须：  
通知投保人/被保险人他们有义务如实告知与保单相关的情况。  
提前14天提供书面通知，表明保险人的续保意图。  
发出续保邀请前，必须先确保贵组织没有其他部门在对索赔进行调查。  
第1步：续保审查警示；  
第2步：续保邀请（续保通知书）；  
第3步：接受续保邀请  
保费支付问题

## 保单失效

如果保险人不准备提出续保邀请，保险人必须通知投保人/被保险人，这类通知通常被称作是续保拒绝书。续保拒绝书中应当载明拒绝原因，因为投保人/被保险人在将来向其他保险人投保时，必须声明自己的保单曾在续保时被拒绝过。

如果上述通知没有像我们讨论的那样按要求的时间和格式发出，那么下一续保期必须提供“免费”保险保障。  
通知客户拒绝续保被看作是良好的业务习惯。

+ 撤销保单的理由

投保人/被保险人未履行最大诚信义务；订立合同时，投保人/被保险人未履行如实告知的义务；在订立合同之前的磋商过程中，投保人/被保险向保险人做出过虚假陈述；投保人/被保险人未遵守合同条款，其中包括保费支付条款；投保人/被保险人在原保险合同的有效其内曾依据该保险合同或者任何其他保险合同提出过欺诈性索赔；有行动或者疏漏发生，而合同规定投保人/被保险人对此有通知保险人的义务，否则基于该行动或者疏漏，合同赋予保险人有拒绝赔付的权利；由于有行动或疏漏发生，改变了保险合同标的物（投保的财产）的状况或者条件。  
因投保人/被保险人未遵守续保条款而生效的所有临时合同、暂保单和“免费”保单全部可以被撤销。

## 宣布保单无效

是一种非常严重的做法，只有在非常罕见的情况下才会使用。该保单如同从未存在过一样，不会对索赔作出赔偿，所有缴纳的保费如数返还。  
如果发现客户在最初购买保险或其后进行续保时未履行如实告知义务，保险人有权宣布保险合同无效。并可以回溯到保险开始的日期与时间，同时退还保费。

## 保险周期的持续

作为一名承保人，在保单的存续周期内，您必须对风险作定期审查。这包括对所有的索赔情况进行监控，并留意风险性质的任何变化。过去，承保和索赔被看作是两个分开的、具有不同职能的部分，但其实它们是属于一个完整及相互关联的风险评估与管理周期中的两个部分。

# 第6章：合同

## 合同的组成部分

意向；要约；协议；承诺；付款；保险标的物的合法性；立约能力；合意；  
另外，保险合同还应包含以下部分：  
最大诚信义务；如实告知义务；

## 签发保险合同

步骤：  
识别客户的需求；从客户那里搜集所有必要资料；在贵公司规定的期限内利用投保申请书搜集资料；适用承保人的条款与条件；核实客户所提供的风险资料；有必要的话，须进行调查。  
实际签发保单：  
按照公司指引、政策和程序确定合同的条款与条件；  
如果投保申请不符合公司指引的要求，应转交给适当的人士作出决策；  
填写并审核合同文件，确保其合乎法律要求；  
按照操作程序检查合同是否准确，是否有任何疏漏；  
按照公司的操作程序签发暂保单、报价或者新的保单，并发送给客户；  
按照操作程序准确地对合同文件进行归档。

## 签发续保通知

第一步是找出需要续保的合同；然后，需要将续保条款告知投保人/被保险人；最后一步是确定续保条款。

## 签发撤销通知

第一步是找出需要撤销的合同；接下来需要处理撤销细节；确认退款或欠款；最后签发相关文件。
